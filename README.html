优化指令

  • 在原本的汇编指令中，mov $0, %R 来给寄存器置 0，占用了7个字节，而使用 xor %R, %R 一样可以置0，只占 3 字节

  • xor %R, %R; inc %R 占用 6 字节，而 mov $1, %R 占用 7 字节
